# AutoSolver Agent 作品简介

AutoSolver Agent 是面向美团 AI Hackathon 命题四“随机配送分配问题”的智能求解作品。它针对随机订单、骑手接单意愿、配送成本和合单约束并存的场景，在有限时间内为每个样例生成合法且尽可能低成本的配送分配方案。

作品采用“正式求解器 + 可视化 Agent”的双层架构：`solver.py` 负责官方评测热路径，保持单文件、无第三方依赖和 10 秒预算内返回；网页端 Agent 展示感知样例、规划策略、试跑候选、自动评估、回退和记忆沉淀的完整过程。系统会根据样例规模、接单意愿和合单结构，自适应选择贪心、多派、匹配、列搜索、流模型和局部改进等策略，并持续维护当前最优合法解。

项目保留架构图、运行截图、测试用例和官方提交记录，便于评委复核。官方记录显示：`avg_score: 706.197`，10 个样例全部成功。本作品重点展示一套可运行、可观察、可验证的 AutoSolver Agent，而不是单纯的静态算法说明。
